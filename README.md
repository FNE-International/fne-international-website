# FNE International Website

Official website for FNE International, a nonprofit organization partnering with communities in developing nations to advance housing, health, and education.

## Tech Stack

- **Framework**: [Astro](https://astro.build)
- **CMS**: [Storyblok](https://storyblok.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Hosting**: [Vercel](https://vercel.com)

## Development

```bash
# Install dependencies
npm install

# Start dev server (https://localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Pushes to `main` automatically deploy to Vercel.

**Live site**: https://fne-international-website.vercel.app
